# Rule-Based AI Chatbot

## Project Overview

This project demonstrates the development of a simple Rule-Based AI Chatbot using Python. The chatbot interacts with users through predefined responses and follows rule-based decision-making logic to generate appropriate replies. The project was completed as part of the DecodeLabs AI Internship program and serves as an introduction to conversational AI and chatbot development.

The chatbot is designed to accept user input, process it using conditional statements, and respond based on predefined rules. This project helped in understanding the fundamentals of Artificial Intelligence, user interaction, and basic chatbot functionality.

---

## Objective

The objective of this project is to build a simple chatbot that can interact with users using predefined responses and demonstrate the basic concepts of conversational AI.

---

## Key Features

* Responds to common greetings
* Handles basic user queries
* Uses predefined rule-based responses
* Supports continuous conversation
* Includes an exit command to end the chat
* Demonstrates decision-making using conditional statements

## Working Principle

The chatbot works by accepting user input and comparing it with predefined conditions using Python's conditional statements (`if`, `elif`, and `else`).

When a user enters a message, the chatbot checks whether the input matches any known commands or keywords. If a match is found, the chatbot returns the corresponding response. If the input is not recognized, a default response is displayed.

The chatbot continues running until the user chooses to exit the conversation.

---

## Project Workflow

1. Accept user input.
2. Process the input using conditional logic.
3. Compare the input with predefined responses.
4. Generate an appropriate reply.
5. Continue the conversation until the user exits.
