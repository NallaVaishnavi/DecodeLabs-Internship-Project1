# Rule-Based AI Chatbot

## Project Overview

This project is a simple Rule-Based AI Chatbot developed using Python as part of my AI internship at DecodeLabs. The chatbot interacts with users by responding to predefined inputs and demonstrates the fundamentals of conversational AI using conditional logic and control flow.

## Key Features

* Responds to common greetings
* Handles basic user queries
* Supports exit commands
* Runs continuously until the user chooses to exit
* Uses rule-based decision-making logic

## Technologies Used

* Python

## Working Principle

The chatbot accepts user input and compares it with predefined commands. Based on the input received, it generates an appropriate response using if-else conditions. If the input does not match any predefined command, the chatbot returns a default response.
